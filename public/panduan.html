Panduan Top Up

Pelajari cara top up game di Malzz Pedia dengan mudah dan cepat.

Cara Top Up Cara Bayar Cek Status FAQ

01 — Cara Top Up

1

Buka Halaman Top Up

Pergi ke halaman Top Up atau klik tombol "Top Up Sekarang" di beranda. Kamu akan melihat daftar semua layanan yang tersedia.

2

Pilih Game & Nominal

Cari game yang ingin kamu top up menggunakan kolom pencarian atau filter kategori. Klik pada nominal yang diinginkan — pastikan sesuai dengan kebutuhanmu.

3

Masukkan ID Akun

Masukkan ID akun game kamu dengan benar. Setiap game memiliki format ID yang berbeda — biasanya berupa User ID atau Server ID. Periksa kembali sebelum melanjutkan.

4

Klik Lanjut Bayar

Setelah semua data benar, klik tombol "Lanjut Bayar". Sistem akan membuat invoice pembayaran secara otomatis.

02 — Cara Bayar

💳 Via QRIS

1. Buka aplikasi dompet digital (GoPay, OVO, Dana, ShopeePay, dll) atau mobile banking.

2. Pilih menu Scan QR atau QRIS.

3. Scan QR Code yang tampil di halaman pembayaran.

4. Pastikan nominal yang tertera sudah benar, lalu konfirmasi.

5. Top up akan diproses otomatis setelah pembayaran dikonfirmasi.

⚠️ Penting: Bayar tepat sesuai nominal yang tertera. Pembayaran kurang atau lebih akan menyebabkan transaksi gagal.

03 — Cek Status Order

1

Simpan ID Transaksi

Setelah membuat order, kamu akan mendapatkan ID transaksi unik. Simpan ID ini sebagai referensi.

2

Buka Halaman Status

Pergi ke halaman Cek Status dan masukkan ID order atau ID transaksi kamu.

3

Lihat Detail Order

Sistem akan menampilkan status terkini dari ordermu — pending, diproses, atau sukses.

04 — FAQ

Berapa lama proses top up? +

Proses top up biasanya berlangsung kurang dari 2 menit setelah pembayaran dikonfirmasi. Pada kondisi tertentu bisa lebih lama.

Bagaimana cara mengetahui ID akun game saya? +

ID akun biasanya ada di menu profil atau pengaturan game. Tiap game berbeda — contohnya Mobile Legends ada di bagian profil, Free Fire ada di pojok kiri atas.

Top up sudah dibayar tapi belum masuk? +

Cek status ordermu di halaman Cek Status. Jika sudah lebih dari 10 menit, hubungi admin dengan menyertakan ID transaksi kamu.

Apakah aman top up di sini? +

Ya, Malzz Pedia menggunakan sistem pembayaran yang aman. Kamu tidak perlu memberikan password akun game — cukup User ID saja.

Bisa refund jika gagal? +

Jika terjadi kegagalan di pihak kami, saldo akan dikembalikan atau akan diproses ulang. Hubungi admin dengan ID transaksi sebagai bukti.
